# Roket V3!
Roket is a game about dodging asteroids. Yes, that's actually it.
Though you can do many more things *whilst* dodging asteroids, like:
- picking up buffs
- using weapons and tools to destroy incoming dangers and heal yourself
- collecting multiple ships with different abilities and upgrading them
- playing through an *expansive* (said no one ever) survival campaign with different conditions and achievements

This version is meant to directly replace the V1 version (https://github.com/smartSmark12/Roket), while presenting more features, a career and a much needed core rewrite

*Notice*: This game is in *very* early development state and is being worked on by a single person (meeee :3), so please be patient, thanks! <3
